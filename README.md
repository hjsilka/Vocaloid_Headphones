# VOCALOID HEADPHONES
![image alt](https://github.com/hjsilka/Vocaloid_Headphones/blob/caadc56b94b6daffc2316fee4b9af7bcdd2448b6/Magnet.webp)

For my Tech Basics 2 final project, I recreated the iconic headphones worn by Hatsune Miku and Megurine Luka
on the cover of the song Magnet. I upgraded the design of the headphones with a fully functional
LED ring that lights up and reacts dynamically to music
